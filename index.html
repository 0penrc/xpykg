xpykg - a package manager for Windows XP

xpykg is a package manager for Windows XP/ReactOS. It downloads files, runs setup.exe, and handles upgrades/uninstallations.

usage:

help: view this page
install: install {pkg}
list: show all the available apps
search: search {pkg}
sync: sync software databases
uninstall: uninstall {pkg}
upgrade: upgrade packages installed using xpkg
version: show xpkg version
